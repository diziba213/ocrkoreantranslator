## Lessons Learned

1. **2B 모델도 도메인 특화 태스크에는 충분하다.** qwen3.5:2b가 VC 용어 설명을 잘 해냈다. 다만 포맷 지시 준수력은 낮아서 프론트에서 보정이 필요했다.

2. **thinking 모드는 반드시 꺼야 한다.** qwen3/3.5의 thinking 토큰이 출력 예산을 잡아먹어 체감 속도가 2~3배 느려졌다. `"think": false`는 `options`가 아니라 payload 최상위에 넣어야 작동한다.

3. **Claude Code Vision은 무료 OCR로 쓸 만하다.** 22장 스크린샷을 Read tool로 직접 읽어 번역하는 것이 외부 OCR API 없이 가능했다. 다만 재무제표 같은 숫자 밀집 표는 원본 이미지 임베드가 더 낫다.

4. **정적 사이트 = 최소 비용 배포.** 로컬 LLM 의존을 JSON으로 전환하면 GitHub Pages만으로 완전 배포 가능. 서버 비용 $0.
